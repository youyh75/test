Hướng dẫn cấu trúc cột trong Sheet "DATA"

1. Số lượng Input (Đầu vào)

  • Cột Y: Số lượng Input.
  • Cột X: Công đoạn Process.
  • Cột Z: Tháng sản xuất.
  • Cột W: Tên Model.
  • Cột U: Loại sản phẩm (Mass / Sample).
  • * Tính tổng TB 3 Process (Tổng Col Y / 3)

2. Số lượng lỗi (Defects)

  • Cột F: Tên Model lỗi.
  • Cột D: Loại sản phẩm lỗi.
  • Cột O: Tháng xảy ra lỗi.
  • Cột G: Công đoạn.
  • Cột J: Tên lỗi.
  • Cột L: Số lượng lỗi.
  • Cột M: Xử lý (Rework / Return line / hủy).

3. Công thức tính (Formulas)

  • 직행율 (FPY): Tính trên tổng lỗi Cột L.
    (Input - Total Lỗi) / Input * 100
  • 불량율 (Defect): Bỏ lỗi "Return line".
    (Lỗi Thực Tế / Input) * 100
  • 수율 (Yield): Lỗi bị "hủy" (Scrap).
    (Input - Scrap) / Input * 100
